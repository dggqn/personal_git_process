# 更新日志

## I18N-0.0.9

- 新增 VS Code 菜单、命令、侧边栏的中英文语言包。
- 插件运行时弹窗会按 VS Code 语言切换：中文界面显示中文，其余语言显示英文。

## SUBMENU-0.0.8

- 编辑器右键改为 `Personal Git Process` 二级菜单，集中展示插件功能。
- 新增当前文件本机块切换入口，可从右键菜单切回本机版本或公共/远程版本。

## CONTEXT-MENU-0.0.7

- 新增编辑器右键菜单：选中代码后可直接保护为本机块。
- 新增编辑器右键菜单：选中代码后可直接更新已有本机块。

## UI-CRUD-0.0.6

- 新增左侧 `Personal Git` 活动栏和 `Local Blocks` 管理视图。
- 支持在视图中查看本机块状态、跳转到代码位置、单块切换本机/公共版本。
- 支持用当前选区更新已有本机块，支持删除本机块并选择是否先恢复公共版本。

## LOCAL-FILTER-0.0.5

- 初始 MVP 版本，提供 `Sync Workspace`、`Safe Push` 和输出日志能力。
